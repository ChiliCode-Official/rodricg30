# 🧩 Portafolio Web Interactivo - Luis Rodrigo (Rodricg30)

Este proyecto es el portafolio web personal e interactivo de **Luis Rodrigo (Rodricg30)**, Diseñador UX certificado por Google y fundador de **Chili Code (Agencia de Software & SaaS)**.

El sitio está optimizado para dispositivos móviles y cuenta con un tema claro (por defecto) y tema oscuro, animaciones por GPU, un Cubo Rubik 3D interactivo en la cabecera, una línea de tiempo con hitos de 2026, cuadrícula de proyectos reales y un formulario de contacto directo con redirección inteligente a WhatsApp.

---

## ✨ Características Premium

- **Cubo Rubik 3D Interactivo**: Construido con transformaciones CSS 3D nativas (sin dependencias pesadas como Three.js), con funciones de rotación táctil/mouse, mezcla (Scramble) y resolución automática (Solve).
- **Alternador de Temas (Claro/Oscuro)**: Tema claro premium por defecto y tema oscuro cyber-minimalista guardado en `localStorage`.
- **Efectos Micro-animados**: Cursor personalizado interactivo, barra de progreso de scroll, canvas de partículas fluidas de colores Rubik y botones magnéticos interactivos.
- **Sección de Ecosistema Chili Code**: Detalle del holding tecnológico B2B, las soluciones SaaS operativas (**GoWash** y **Shift**), Chili Code Express (webs en 48h) y hardware NFC (TapStar, PlatePost).
- **Trayectoria Profesional**: Línea de tiempo de hitos por trimestre de 2026 (Ingeniería en Sistemas UVM, Google I/O 2026, Google México, consultoría de IA para Goodyear/Swissbrand/Zoho) e historial laboral.
- **Redirección Directa a WhatsApp**: Formulario de contacto optimizado para móviles B2B que redirecciona al usuario con un mensaje estructurado y pre-llenado a WhatsApp.
- **SEO & Metadatos Completos**: Optimización para buscadores e integración de JSON-LD y OpenGraph.

---

## 🛠 Stack Tecnológico

- **HTML5 Semántico**
- **CSS3 Moderno** (Vanilla CSS, variables CSS personalizables, flexbox, 3D grids y animaciones aceleradas por GPU)
- **Vanilla JavaScript** (Lógica de física del cubo 3D, canvas de partículas, intersection observers y lógica de temas)

## 🚀 Despliegue en GitHub Pages

Este portafolio es 100% estático y está optimizado para cargarse instantáneamente en GitHub Pages.

1. Añadir cambios a tu repositorio git:
   ```bash
   git add .
   git commit -m "Redesign portfolio: add Chili Code 2026 content and theme toggle"
   ```

2. Sube los cambios a la rama principal de tu repositorio en GitHub:
   ```bash
   git push origin main
   ```

3. El sitio web estará disponible en vivo de forma inmediata en la URL de GitHub Pages asociada a tu cuenta.
